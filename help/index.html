Glaceon command list

Square brackets mean an optional argument, Parentheses mean a required argument

Command Usage Description Required permission
help (no arguments) Links to this page None
clean (number of messages) [member] bulk-deletes a certain number of messages. If a user is specified, it only deletes messages from that user. Per-channel Manage Messages
warn (member) (reason) Send a user an official warning, ususally for breaking the rules. Server-wide Manage Messages
mute (member) [duration] (reason) Mutes a member so they cannot send messages for a certaion duration, and notifies them of the reason. Valid duration entries are #t where t can be s (seconds), m (minutes), h (hours), d (days), w (weeks), or y (years). Server-wide Manage Messages
unmute (member) Undoes the actions of the mute command. Server-wide Manage Messages
kick (member) [reason] Removes a member from your server until they rejoin with another invite link. Kick Members
ban (member) [duration] (reason) Removes a member and bans them so they cannot rejoin, and notifies them of the reason. Valid duration entries are #t where t can be s (seconds), m (minutes), h (hours), d (days), w (weeks), or y (years). Ban Members
pardon (member) Removes a ban from a user so they can rejoin if they have another invite link. Ban Members
lock [channel] Locks a channel so that users without Manage Messages given by another role will not be able to chat. Manage Channels
unlock [channel] Undoes the lock command Manage Channels
