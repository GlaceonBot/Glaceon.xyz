Glaceon command list

Square brackets mean an optional argument, Parentheses mean a required argument

Command Usage Description
help (no arguments) Links to this page
clean (number of messages) [member] bulk-deletes a certain number of messages. If a user is specified, it only deletes messages from that user.
warn (member) (reason) Send a user an official warning, ususally for breaking the rules.
